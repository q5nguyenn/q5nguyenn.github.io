CSS Grid Layout

CSS Grid Layout, một phương pháp sắp xếp và cấu trúc các phần tử trên trang web trong một lưới hai chiều (dòng và cột). CSS Grid Layout cho phép bạn tạo ra các khu vực (areas) trong lưới và điều chỉnh vị trí của các phần tử trong các khu vực đó một cách linh hoạt và mạnh mẽ.
